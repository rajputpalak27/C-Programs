Simple multiple problem
You are given 2 numbers N and M.You have to find the smallest number which when multiplied to N makes it a multiple of M.

Input:

The first line of input contains  T, which represents the number of test cases.Each of the next T lines contains two numbers, Nand M .

Sample Input :

2
4 7
18 6

Output :
    For each test case
    Print the required answers.

    Sample Output :

7
1
